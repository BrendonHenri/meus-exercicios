modelos de códigos

  1. <kbd> → elemento define a entrada do teclado

  2. <samp> → elemento define a saída de amostra de um programa de computador

  3. <code> → elemento define um pedaço de código de computador

  4. <var> → elemento define uma variavel na programaçao ou em uma expreção matemática

  5. <pre> → elemento define o texto pré-formatado
